# 🐾 Smart Paws: AI Veterinary Assistant + Vaccination Tracker

Smart Paws is an intelligent AI assistant and vaccination tracker for pet owners. Built using **Gradio**, it helps users ask questions about their pet's symptoms, nutrition, breed-specific traits, and vaccination schedules.

---

## ✨ Features

- 🤖 **AI Assistant** powered by OpenAI for:
  - Symptom checking with detailed advice.
  - Nutrition guidance for cats and dogs.
  - Breed-specific information.
- 💉 **Vaccination Record Tracker**:
  - Save and view your pet’s vaccination history.
- 🧩 User-friendly interface using Gradio.
- 🌐 Built-in support for local pet context (e.g., Pakistan-based food options).
